📋 替换图标步骤

  1. 点击上方"下载所有尺寸图标"按钮
  2. 解压下载的 ZIP 文件
  3. 将各尺寸的 PNG 文件复制到 Flutter 项目的 android/app/src/main/res/mipmap-xxx/ 目录
  4. 运行 flutter clean && flutter build apk --release 重新打包
  5. 安装新 APK 到手机

目录对照表：

  • mipmap-mdpi/ → 48×48
  • mipmap-hdpi/ → 72×72
  • mipmap-xhdpi/ → 96×96
  • mipmap-xxhdpi/ → 144×144
  • mipmap-xxxhdpi/ → 192×192
